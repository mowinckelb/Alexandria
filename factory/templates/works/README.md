# Works

*Your personal writing and curation — essays, aphorisms, meditations, quotes, reading material. The publishable corpus that's yours but distinct from raw vault.*

*Different from `vault/` (raw, append-only, everything captured), `ontology/` (in-progress thoughts), and `constitution/` (crystallised beliefs). Works is the curated collection — what you've consciously written or chosen to keep.*

*A scholar's library. Use freely.*

## systems/

Reusable Alexandria machinery the Author invents: prompts, filters, extraction moves, rituals, publishing formats, workflows.

If a system is only for you, keep it here. If it could help other Authors, make a stripped contribution candidate and ask whether to submit it to the Alexandria GitHub repo. Once it has a GitHub path, use that path as its marketplace module ID: `github:<owner>/<repo>#<path-without-.md>`. Until then, use `local:<github-login>/<slug>` in `~/alexandria/.call_manifest`.
