# Library

Local cache of what you may share with the protocol.

- `public/shadow_proposal.md` — Alexandria's recommended public shadow draft. Review/edit freely. Not published.
- `public/shadow.md` — your approved public shadow. This is safe-for-strangers material and is published automatically to satisfy the monthly file obligation.
- `paid/` and `invite/` — later Library tiers.

Final files are consent. Drafts/proposals are not.
